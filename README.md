## ‚öôÔ∏è Setup Tutorial: How to Import Subscriptions to CFA
The setup process for Clash for Android is straightforward and compatible with all major proxy service providers.

### Step 1: Import Subscription Profile
1. Open the Clash for Android app and tap on **Profiles**.
2. Tap the **+ (Plus)** icon in the top right corner.
3. Select **URL (Import from URL)**.
4. Name your profile (e.g., "My Subscription") and paste your **Clash Subscription Link** into the URL field.
5. Tap the **Save (Disk)** icon. Wait for the configuration to download.
6. Go back to the profile list and tap to select the new profile.

### Step 2: Start the Proxy Service
1. Return to the App's main dashboard.
2. Tap the **Stopped** button in the center (it will change to **Running**).
3. Grant the **VPN Connection Request** if prompted by the system.
4. You are now connected!

### Why is my subscription failing to import?
1. **Format:** Ensure the link is a **Clash YAML** format, not a plain SSR or V2Ray link.
2. **Connectivity:** Try downloading while connected to mobile data or another temporary proxy if your current network blocks the server.
3. **Time Sync:** Ensure your phone's system time is accurately synced with network time.
